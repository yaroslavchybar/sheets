# **App Name**: SheetFlow

## Core Features:

- Sheet Data Display: Display the Google Sheet data in a well-designed table format.
- Secure Authentication: Implement secure authentication using Google Sheets API and NextAuth to protect data.
- Role-based Access Control: Differentiate user access levels, granting admins full control while members have restricted permissions.
- Interactive UI: Provide an intuitive user interface with Shadcn UI components for managing and interacting with Google Sheet data.
- Google Sheet data import: Users can import data from the google sheet (https://docs.google.com/spreadsheets/d/1lqzwL5JPaH8PoWXYCuEHk7ewGo74aPjPBQrVHr-72RE/edit?usp=sharing).
- Sheet Interaction: Allow members and admins to interact with the sheet; UI checkbox changes reflect in the Google Sheet.

## Style Guidelines:

- Primary color: #F8F9FA - Very light gray, almost white, for main backgrounds.
- Secondary color: #E9ECEF - Light gray for subtle UI elements and separators.
- Background color: #DEE2E6 - Medium-light gray for secondary backgrounds or containers.
- Text color: #212529 - Very dark gray, almost black, for primary text providing high contrast.
- Body and headline font: 'Inter', a sans-serif typeface.
- Crisp, modern icons from a library like 'Lucide' for clear and intuitive navigation.
- Clean, spacious layout with a focus on data presentation using Shadcn UI's grid and table components.
- Subtle transitions and feedback animations on user interactions, leveraging Shadcn UI's animation utilities.
